## How to Run
1. Clone the repo
   ```sh
   git clone https://github.com/aliazimoshan/BluBank.git
   ```
2. Install all the packages by typing
   ```sh
   flutter pub get
   ```
3. Run the App
   ```sh
   flutter run
   ```
